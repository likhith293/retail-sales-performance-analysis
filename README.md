# Retail Sales Performance Analysis

## Overview

This project analyzes retail sales data to uncover business insights related to product categories, customer segments, regional performance, sales trends, and shipping efficiency.

The project also demonstrates a basic machine learning workflow using Linear Regression for sales prediction.

## Objectives

* Clean and preprocess retail sales data
* Perform exploratory data analysis (EDA)
* Identify key sales trends and patterns
* Generate business insights
* Build a basic machine learning model

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Data Cleaning

* Handled missing values
* Removed duplicate records
* Converted date columns
* Created new features:

  * Order Month
  * Delivery Days

## Key Insights

* Technology generated the highest sales revenue.
* The West region recorded the highest sales.
* Consumer customers contributed the largest share of revenue.
* Sales peaked during November and December.
* Same Day shipping delivered the fastest service.

## Machine Learning

A Linear Regression model was developed to predict sales using engineered features from the dataset.
